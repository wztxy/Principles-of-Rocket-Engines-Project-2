
## 读入文件格式说明
第$1$行三个整数 $P$, $M$, $N$, $L$分别代表推进剂组元数，元素数，平衡组分数，凝相平衡组分数

接下来$P$行，每行$M$个整数$St_{i,j}$，分别表示$1 mol$第$i$种组元含有第$j$种元素$St_{i,j} mol$

接下来一行$P$个浮点数$g_i$，代表第$i$种组元的质量分数

接下来一行$M$个整数$Atom_j$，分别代表第$j$种元素的原子量（推荐保留4位以上小数），原子量从小到大排序，$St$矩阵要对应

接下来$N$行，每行$7$个浮点数，表示第$i$种平衡组分的$7$系数物性参数，要求凝相组分在前，气相组分在后。

接下来一个$M \times N$矩阵$\{A_{i,j}\}$，表示$1mol$第$j$种平衡组分含第$i$种元素的摩尔数。要求元素排列顺序和平衡组分排列顺序与前面相同。

接下来一行$1$个浮点数，表示以大气压（$atm$）为单位的燃烧室压力$p_c$
接下来一行$3$个浮点数，表示最低试算温度（$K$），最高试算温度（$K$），温度步长（建议取大于$25K$）

接下来一行$N$个浮点数$\{c_{j}\}$，表示给定的平衡组分迭代初始值（建议按照化学方程式确定平衡组分，主要组分按照方程式填入，次要组分写小量）

接下来一个浮点数，表示初始推进剂总焓（要提前查值计算）

接下来一个整数id

* id = 1，给定截面压强进行喷管计算，使用平衡流动计算。
* id = 2， 给定截面温度进行喷管计算，使用冻结流动。
