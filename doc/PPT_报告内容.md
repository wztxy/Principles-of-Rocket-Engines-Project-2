## Part 1、项目概述

### 项目目标
- 实现 **美国 SLS 登月火箭发动机热力性能参数计算**
- 支持 **LOX/LH2（液氧/液氢）** 和 **LOX/CH4（液氧/甲烷）** 两种推进剂
- 基于 **最小吉布斯自由能法** 求解化学平衡
- 采用 **NASA 9系数多项式** 计算热力学性质

### 技术栈
- **核心算法**: C 语言（纯 C99，无依赖）
- **图形界面**: Qt 6.8+ / C++17
- **构建系统**: qmake（本地）/ CMake（CI）
- **跨平台**: macOS (arm64/x64)、Windows (x64)、Linux (x64)

### 支持的发动机预设
| 发动机 | 推进剂 | 燃烧室压强 | 混合比 O/F | 真空比冲 |
|--------|--------|------------|------------|----------|
| RS-25 (SSME) | LOX/LH2 | 20.64 MPa | 6.03 | 452 s |
| RL-10B2 | LOX/LH2 | 4.36 MPa | 5.88 | 466 s |
| J-2X | LOX/LH2 | 9.2 MPa | 5.5 | 448 s |
| SpaceX Raptor | LOX/CH4 | 30.0 MPa | 3.6 | 363 s |
| YF-77 | LOX/LH2 | 10.2 MPa | 5.5 | 430 s |
| Vulcain-2 | LOX/LH2 | 11.7 MPa | 6.1 | 434 s |

## Part 3、研究方法——计算原理

### 3.1 燃烧室计算流程

**输入**: 燃烧室压力 $p_c$、推进剂组成、初始焓 $H_{pre}$

**步骤**:

1. **假定化学式计算** (`calc_assumed_formula`)
   - 将推进剂质量分数转换为元素摩尔数 $N_k$
   - 公式: $N_k = \sum_{i} g_i \cdot B_{ik}$，其中 $B_{ik} = St_{ik} \cdot 1000 / M_i$

2. **温度扫描 + 最小吉布斯自由能法**
   - 在 $T_{min}$ 到 $T_{max}$ 范围内扫描（步长 25K）
   - 对每个温度调用 `min_gibbs_solve()` 求解平衡组分
   - 使用 **拉格朗日乘子法** 求解约束优化问题

3. **焓守恒插值**
   - 计算每个温度点的总焓: $H_{total} = \sum_j n_j \cdot H_j(T)$
   - 寻找满足 $H_{total} = H_{pre}$ 的温度点
   - 线性插值确定燃烧温度 $T_c$ 和对应组分

**输出**: 燃烧温度 $T_c$、平衡组分 $n_j$、热力参数 (γ, c*, Cp, Cv, ...)

### 3.2 最小吉布斯自由能法详解

**目标函数**: 最小化总吉布斯自由能
$$G = \sum_{j=1}^{N} n_j \mu_j = \sum_{j=1}^{N} n_j \left[ g_j^0(T) + RT\ln\frac{n_j p}{n_g} \right]$$

**约束条件**: 元素守恒
$$\sum_{j=1}^{N} a_{kj} n_j = N_k, \quad k = 1, 2, ..., M$$

**拉格朗日函数**:
$$L = G + \sum_{k=1}^{M} \lambda_k \left( N_k - \sum_{j=1}^{N} a_{kj} n_j \right)$$

**线性方程组** (核心迭代方程，公式28-29):

$$\begin{bmatrix} R & \alpha & A \\ \alpha^T & 0 & 0 \\ A^T & 0 & 0 \end{bmatrix} \begin{bmatrix} \pi \\ \Delta\ln n_g \\ \Delta n_s \end{bmatrix} = \begin{bmatrix} b \\ 0 \\ c \end{bmatrix}$$

其中:
- $R_{kl} = \sum_j a_{kj} a_{lj} n_j$ (R矩阵)
- $\alpha_k = \sum_j a_{kj} n_j$
- $\pi_k = \lambda_k / RT$ (拉格朗日乘子)

### 3.3 喷管计算流程

**输入**: 出口压强 $p_e$、燃烧室结果

**步骤**:

1. **估算出口温度初值**
   - 使用等熵关系: $T_e^{(0)} = T_c \cdot (p_e/p_c)^{(\gamma_s-1)/\gamma_s}$

2. **温度扫描求等熵点**
   - 从高温向低温扫描
   - 对每个温度点求解平衡组分
   - 计算总熵: $S = \sum_j n_j S_j - R \sum_j n_j \ln(n_j/n_g) - R n_g \ln p$

3. **等熵插值**
   - 寻找满足 $S_{exit} = S_{chamber}$ 的温度点
   - 线性插值确定出口温度和组分

4. **性能参数计算**
   - 出口速度: $v_e = \sqrt{2(H_c - H_e)}$ (能量守恒)
   - 比冲: $I_{sp} = v_e / g_0$
   - 马赫数: $Ma = v_e / a_e$
   - 推力系数: $C_F = \Gamma \sqrt{\frac{2\gamma}{\gamma-1}[1-(p_e/p_c)^{(\gamma-1)/\gamma}]} + \frac{p_e}{p_c} \cdot \epsilon$

**输出**: 出口温度 $T_e$、出口组分、比冲 $I_{sp}$、马赫数 $Ma$、推力系数 $C_F$

### 4.3 NASA 9系数多项式格式

**来源**: NASA Glenn Research Center CEA 数据库 (RPA-4.0.7/resources/thermo.inp)

**公式** (有效范围: 200K - 20000K):

$$\frac{C_p}{R} = a_1 T^{-2} + a_2 T^{-1} + a_3 + a_4 T + a_5 T^2 + a_6 T^3 + a_7 T^4$$

$$\frac{H}{RT} = -a_1 T^{-2} + a_2 \frac{\ln T}{T} + a_3 + \frac{a_4 T}{2} + \frac{a_5 T^2}{3} + \frac{a_6 T^3}{4} + \frac{a_7 T^4}{5} + \frac{b_1}{T}$$

$$\frac{S}{R} = -\frac{a_1}{2T^2} - \frac{a_2}{T} + a_3 \ln T + a_4 T + \frac{a_5 T^2}{2} + \frac{a_6 T^3}{3} + \frac{a_7 T^4}{4} + b_2$$

**为什么选择 NASA 9系数而非 7系数**:
- 7系数多项式在高温 (>3000K) 时误差爆炸 (T⁴, T⁵ 项)
- NASA 9系数采用分段 + 负指数项，高温稳定性更好
- 经验证，H₂O 在 3500K 时误差 < 0.1%

### 4.4 平衡比热与偏微分计算

**冻结比热** (公式35-37):
$$c_{p,f} = \sum_j n_j C_{p,j}(T), \quad c_{v,f} = c_{p,f} - n_g R, \quad \gamma_f = c_{p,f}/c_{v,f}$$

**平衡比热** (公式30-32):
$$c_p = c_{p,f} + \sum_j n_j H_j \cdot \frac{\partial \ln n_j}{\partial \ln T}\bigg|_p \cdot \frac{1}{T}$$

**等熵指数** (公式33):
$$\gamma_s = -\gamma \cdot \left(\frac{\partial \ln v}{\partial \ln p}\right)_T^{-1}$$

**偏微分求解**: 通过求解线性方程组获得 $\partial \ln n_j / \partial \ln T$ 和 $\partial \ln n_g / \partial \ln T$

## Part 5、难点突破与创新点

### 5.1 迭代收敛性保证

**问题**: 最小吉布斯自由能法在远离平衡点时可能不收敛

**解决方案**:
1. **智能初值设置**: 根据推进剂类型预设合理初始组分
   ```c
   /* LOX/LH2: H, H2, H2O, O, O2, OH */
   c_init = {1e-3, 10.0, 30.0, 1e-4, 1e-3, 1.0};
   /* LOX/CH4: H2O, H2, OH, H, CO2, CO, O2, O */
   c_init = {35.6, 0.1, 0.01, 1e-5, 5.6, 6.0, 1e-5, 1e-6};
   ```

2. **连续迭代**: 使用上一温度点结果作为下一点初值

3. **数值保护**: 组分下限 `MIN_MOLE_FRAC = 1e-16`，防止对数发散

### 5.2 高温热力数据精度

**问题**: 火箭燃烧室温度 3500-4000K，传统多项式外推误差大

**解决方案**:
- 采用 NASA 9系数分段多项式 (3个温度区间: 200-1000K, 1000-6000K, 6000-20000K)
- 数据来源可追溯: Cox,1989 (H₂O); Gurvich,1978 (H₂, O₂); Ruscic,2002 (OH)

**验证**: H₂O 在 298K 生成焓 = -241826 J/mol，与 NIST 标准值完全吻合

### 5.3 GUI 与计算引擎分离

**设计原则**: 
- 核心算法用纯 C 实现，无任何外部依赖
- Qt GUI 仅负责参数输入和结果显示

**好处**:

- 跨平台移植简单 (仅需重写 UI)
- 算法可嵌入其他系统

### 5.4 输运性质计算

**创新点**: 实现了完整的输运性质计算 (公式41-47)
